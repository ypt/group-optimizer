Group Optimizer
===

Form optimal groups of items, people, etc. using a genetic algorithm.


Dependencies
---

- [Node.js](https://nodejs.org/) + npm

Getting started
---
After cloning this repo into a directory:

```
$ npm install
$ node app
```

Most of the logic should reside in the files in the `/lib` directory.
